SECTION E — ARCHITECTURE NOTES
-------------------------------
API Response Flow:
  Backend sendSuccess() → { success: true, data: payload }
  Frontend ApiClient.request() → unwrapApiResponse() → payload directly
  Never check response.success on service results.

Auth Flow:
  Login → generateAccessToken (15min) + generateRefreshToken (7d) → HTTP-only cookies
  Cookie: HttpOnly, SameSite=Lax, Secure (prod), refresh restricted to /api/auth
  RBAC: deterministic from JWT payload, no DB lookup

Cache Strategy:
  critical (dashboard): 30s stale, refetch on focus
  static (catalog): 5min stale, no refetch on focus
  polling (admin stats): 60s interval minimum
